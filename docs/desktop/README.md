## 🚀 三个命令开始

### 1. 开发模式
```bash
.\start-tauri-dev.ps1
```

### 2. 打包应用
```bash
cd web
pnpm run build:tauri:full:debug
```

### 3. 运行打包后的应用
```bash
.\web\src-tauri\target\debug\app.exe
```
