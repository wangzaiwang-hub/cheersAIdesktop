# CheersAI 桌面应用 - 快速开始

## 🎯 三种使用方式

### 1️⃣ 开发模式（推荐用于开发）

**一键启动所有服务 + Tauri 桌面应用**

```bash
.\start-tauri-dev.ps1
```

**特点**：
- ✅ 功能 100% 完整
- ✅ 热重载支持
- ✅ 立即可用
- ⏱️ 启动时间：1-2 分钟

---

### 2️⃣ 生产打包（推荐用于分发）

**打包成独立可执行文件**

```bash
cd web

# Debug 版本（快速测试）
pnpm run build:tauri:full:debug

# Release 版本（生产发布）
pnpm run build:tauri:full
```

**打包产物**：
```
web/src-tauri/target/release/
├── app.exe              # 可执行文件
└── bundle/
    ├── msi/            # Windows 安装包
    └── nsis/           # Windows 安装程序
```

**特点**：
- ✅ 独立可执行文件
- ✅ 可分发给用户
- ✅ 桌面应用体验
- ⚠️ 需要 Node.js 运行时
- ⏱️ 打包时间：5-10 分钟

---

### 3️⃣ Web 部署（推荐用于生产）

**部署为 Web 应用**

```bash
# 启动数据库
.\start-database.ps1

# 启动后端
.\start-backend.ps1

# 启动前端
cd web
pnpm dev
```

**访问**：http://localhost:3000

**特点**：
- ✅ 标准 Web 应用
- ✅ 无需安装
- ✅ 多用户访问
- ⏱️ 启动时间：30-60 秒

---

## ❓ 常见问题

### Q: 打包后的 .exe 能直接运行吗？
A: 可以，但用户系统需要安装 Node.js。建议在安装包中包含 Node.js 运行时。

### Q: 打包需要多长时间？
A: 
- 首次打包：10-15 分钟（下载依赖 + 编译）
- 后续打包：5-8 分钟（仅编译）

### Q: 打包后的文件有多大？
A: 
- 可执行文件：~5MB
- 加上 Node.js：~55MB
- 完整安装包：~60-80MB

### Q: 支持哪些平台？
A: 
- ✅ Windows（已测试）
- ⏳ macOS（需要测试）
- ⏳ Linux（需要测试）

### Q: 如何优化启动速度？
A: 
1. 使用 Release 版本
2. 预热 Next.js 服务器
3. 优化 Node.js 启动参数
